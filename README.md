# Финотчёт ВБ — разрез по артикулам

Одностраничный статический сайт: загружаете детализацию еженедельного отчёта Wildberries (xlsx) — получаете разбор по артикулам: выручка, логистика, хранение, штрафы, прочие удержания, реклама, себестоимость, налог, прибыль и маржа.

Всё считается в браузере, файлы никуда не отправляются. Обновление страницы стирает данные (настройки налогов сохраняются в localStorage).

## Использование

1. **Детализация отчёта** — xlsx из кабинета ВБ («Финансовые отчёты → Детализация») / Лучше использовать WBin для выгрузки фин.отчетов. Доступен для скачивания в Chrome Web Store.
2. **Себестоимость** — xlsx/csv: `Артикул ВБ | Себестоимость за штуку`.
3. **Реклама** — xlsx/csv: `ID РК | Артикул ВБ | Затраты`.
4. Кнопка «Настроить налоги» — режим (УСН «Доходы» / УСН «Д−Р» / НПД / ОСН) и ставка.
5. «Скачать XLSX» — экспорт собранной таблицы.

## Стек

Без сборки: HTML + CSS + JS. Локальные библиотеки в `lib/`:
- [AG Grid Community](https://www.ag-grid.com/) 32.3.9 — таблица;
- [SheetJS](https://sheetjs.com/) 0.20.3 — чтение/экспорт xlsx и csv.

В `Python/finreport.py` — исходная серверная версия (Flask), логика та же.
